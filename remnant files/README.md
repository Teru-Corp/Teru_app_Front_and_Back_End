# Remnant Files

This folder contains the old, unused files from the previous Expo React Native mobile application architecture. 
Since the project has migrated to a standard Vite React Web Application (located in the `web-frontend` directory), these files are no longer needed for the current application to run. 

They have been kept here for reference purposes, to ensure no code or assets are permanently lost during the transition.

### Contents:
- **`app/`, `components/`, `constants/`, `context/`, `hooks/`, `scripts/`, `types/`, `utils/`, `api/`**: Old Expo application source code and hooks.
- **`assets/`**: Images and fonts used by the mobile app.
- **`node_modules/`, `package.json`, `package-lock.json`**: Dependencies for the Expo app.
- **Config files**: `app.json`, `babel.config.js`, `metro.config.js`, `tsconfig.json`, `eslint.config.js` used by Expo and Metro bundler.
- **`check_db_temp.js`**: An old temporary database check script.

If you ever need to reference how the old mobile app was built, you can look inside this folder!
